# Make-демо: «как просто это собирается»

> Это **не production-инструмент**. Это демонстрация со сцены, что архитектуру инфо-помощника можно собрать в визуальном low-code инструменте за пару минут.
>
> Production-вариант → [../skill/](../skill/) — Claude Skill с cron, дедупом и Telegram-доставкой. Его ставь себе в работу.

## Зачем это в репо

На сцене Дмитрий показывает Make-сценарий из 5 нод — это самый наглядный способ объяснить, **какие куски соединяются с чем**:

```
[Cron 9:00] → [Iterator: clients] → [Perplexity: news] → [Claude: summarize] → [Telegram: send]
```

Если хочешь повторить демо у себя на ноуте — импортируй [make-blueprint.json](make-blueprint.json) и подставь ключи. Это займёт ~4 минуты.

## Импорт за 4 минуты

1. **make.com** → войди → **Create a new scenario**
2. Меню (⋯) → **Import Blueprint** → [make-blueprint.json](make-blueprint.json)
3. Открой каждую ноду → подставь:
   - Perplexity API key (в HTTP-ноде, заголовок Authorization)
   - OpenAI/Anthropic connection (в саммаризаторе)
   - Telegram bot token + chat_id (в последней ноде)
4. В Iterator → подставь одну свою компанию: `[{"name": "...", "website": "..."}]`
5. **Run once** → проверь Telegram

## Почему для продакшена лучше Claude Skill, а не Make

| | Make | Claude Skill |
|---|---|---|
| Стоимость на 50 клиентов/день | $9 Make + $30 Perplexity + $5 OpenAI = ~$45/мес | Только Anthropic API ~$10/мес |
| Дедуп между запусками | Городишь через Google Sheet | Встроена |
| Адаптация тона под клиента | Жёстко в промпте | Skill читает контекст из `clients.yaml` |
| Запуск без интернета (локально) | Невозможен | Возможен |
| Кому ставить | Менеджер, который ни разу не открывал IDE | Команда хотя бы с одним AI-юзером |

Если ты не из IT-команды и Claude/Cursor для тебя пока экзотика — **Make это нормальный путь**. Просто знай, что более масштабируемый вариант лежит рядом, и когда команда дозреет — мигрируешь.
